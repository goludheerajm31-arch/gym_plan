# Project Statement

## Project Title
**Fitness Tracker Web App**

## Purpose
This project is aimed at creating a simple and easy-to-use web application that helps users plan and keep track of their daily workouts. It also allows users to create custom exercises and monitor their water intake. The goal is to encourage healthy habits while keeping the app lightweight and easy to use, without requiring complex setups or databases.

## Objectives
- **Daily Workout Planning:** Users can create and view their daily workout routines, including sets, reps, and optional notes.  
- **Custom Workouts:** Users can add their own exercises to match their personal fitness goals.  
- **Water Intake Tracking:** Users can log their daily water consumption and stay on top of their hydration.  
- **Clean and Simple Interface:** The app is designed to be user-friendly, responsive, and visually simple.  
- **Lightweight Data Storage:** All data is stored locally in a JSON file, keeping the app fast and portable.

## Conclusion
The Fitness Tracker Web App is a beginner-friendly project that demonstrates practical use of Flask, HTML, CSS, and JSON. It provides a functional and clean interface, making it easy for users to manage their fitness routines and monitor daily water intake effectively.
